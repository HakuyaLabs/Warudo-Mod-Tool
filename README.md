# Warudo Mod Tool
A modding tool for Warudo.

## Installation
To install the Warudo Mod Tool, turn off `Assembly Version Validation` in `Player Settings`.

Then add the following line to your Unity project's Package Manager "Add package from git URL" option:

```
https://github.com/HakuyaLabs/Warudo-Mod-Tool.git
```

If any dialog ask for confirmation, accept it.

Now you can use the Warudo Mod Tool in your Unity project!
